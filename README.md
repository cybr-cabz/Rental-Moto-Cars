# Rental-Moto-Cars
A Vehicle rental platform develop by Dan Ranier Cababan
This platform use to rent a vehicle through online but still this platform need more features 
It is just a test-project website for renting online using Streamlit to submit to the Subject CS Elective 3 
Class Adviser: Sir Randy Bello  

Before Using it make sure naka dark mode ka salamat

Features:
Item Specs: Detailed condition and engine reports for every vehicle.
Item Image: The Picture of the vehicle
Pop-up modal: Like a warning before you can rent you need to sign-up or log-in

It has Navs such as:
Home: The Dashboard Pages of the Website
About: The Info Page of the Website
Contact: The Communication Info Page of the Website
Services: The Offer Services of the Website such as Verified Vehicle Condition, Privacy & Data Protection, Secure Connected Experiences, Enhance Site Security, and Expert Road Support. 

Tech/Lang Used:
Python
Streamlit
